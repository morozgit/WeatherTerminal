# WeatherTerminal
## Выводит погоду для Лондона, Шереметьево и Череповца

Для запуска скрипта 

1. Установите библиотеку requests
```python
pip install requests
```
1. Клонируйте репозиторий

1. В терминале пройти в дерикторию со скриптом и ввести команду
```python
python3 main.py
```
