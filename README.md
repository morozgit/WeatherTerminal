# WeatherTerminal
## Выводит погоду для Лондона, Шереметьево и Череповца

Для запуска скрипта 

1. Установите библиотеку requests
```python
pip install requests
```
2. Клонируйте репозиторий

3. В терминале пройти в дирикторию со скриптом и ввести команду
```python
python3 main.py
```
